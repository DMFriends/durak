# Durak

A popular Russian shedding-type card game where the goal is to get rid of all your cards to avoid being the last player with cards remaining, who is deemed the "durak" (fool). Played with a 36-card deck (6-Ace), players attack the defender with cards, which must be beaten by higher cards of the same suit or a trump card.

## Rules

Read the full rules [here](https://playjoy.com/en/durak/rules/).

## Running the program

Assuming you have Java installed:
1. Clone this repository
2. Run `java Durak.java` in the root directory
